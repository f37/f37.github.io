Neurocat module¶

A modest software toolbox for creating and training Neural Networks in tensorflow and for handling of data input.

class Neurocat.Loading(nepoch, label='')¶

Bases: object

Visualisation of loading process in as graphic interface for the terminal. Gives information of the state of the process and linearly approximates remaining time for calculation.

Example:

train the network
#################
Loading 25000 Epochs
Approx. time remaining: 0.27 minutes
Parameters:
  • nepoch – Total number of calculations stages
  • label – name of calculation process
